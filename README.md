# Smart Safety Vision

Smart Safety Vision is an educational computer vision project built with Python, OpenCV, and Streamlit.

The goal of this project is to learn computer vision step by step and build a practical safety detection dashboard.

## Project Goals

- Learn Python project structure
- Learn Git and GitHub workflow
- Learn basic computer vision with OpenCV
- Build an interactive dashboard with Streamlit
- Extend the project with YOLO object detection

## Current Features

- Basic Python project setup
- Virtual environment configuration
- First OpenCV image generation
- Output image creation with text and rectangle drawing

## Tech Stack

- Python 3.10
- OpenCV
- NumPy
- Pillow
- Streamlit
- Git
